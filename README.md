# WEBSITE NAME 
HomeNest - A Real Estate Listing Portal
# LIVE SITE LINK
[HomeNest](https://home-nest-d37.netlify.app/)
# FEATURES
* Users can add properties to sell.
* Users can browse all listed properties with filters and sorting options.
* Users can Sign in/ Sign up to get access of all available features.
* Each property shows detailed information including price, location, and agent contact.
* Responsive design for seamless experience across all devices
